# Deskripsi program

Sebuah program PHP sederhana untuk mengubah teks dengan EYD ke ejaan lama (ejaan van Ophuijsen).

## Aturan Ejaan van Ophuijsen

---------------------------------------
	ejaan van ophuijsen	|		EYD	
---------------------------------------
			dj			|		j
			tj			|		c
			j			|		y
			nj			|		ny
			sj			|		sy
			ch			|		kh
			oe			|		u
		glottal stop(')	|		k
---------------------------------------
---------------------------------------
